# Math Pattern

Math problems require number theory, geometry, or algebraic manipulation. They cover modular arithmetic, digit manipulation, GCD/LCM, prime numbers, and coordinate geometry.

## When to Use
- Number base conversions (Roman numerals, binary)
- Digit manipulation (plus one, multiply strings)
- Square root / power calculations
- Trailing zeros in factorials
- Point/line geometry problems

## Key Techniques
1. **Modular arithmetic**: `%` operator for digit extraction
2. **Digit manipulation**: Process digits from right to left
3. **Mathematical formulas**: Direct computation (trailing zeros = n/5 + n/25 + ...)
4. **Coordinate geometry**: Slope calculation, collinearity checks
5. **String math**: Handle large numbers as strings

## Common Patterns
- Roman numeral conversion (map + subtract when smaller before larger)
- Factorial analysis (count factors of 5 for trailing zeros)
- Power computation (fast exponentiation)
- Point on line (slope equality with cross product)
